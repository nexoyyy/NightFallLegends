# NightFall Legends - Game Website

This project is a website for the fictional game "NightFall Legends". It's an interactive showcase with a modern and attractive interface to present the game and provide download options.

## Features

- Modern and responsive design
- Dynamic animations and visual effects
- Interactive game trailer animation
- Game features presentation sections
- Image gallery
- Download section with links for different platforms
- Intuitive and attractive user interface

## Project Structure

- `index.html` - Main site structure
- `styles.css` - Styles and layout
- `script.js` - Interactive features and animations
- `video.js` - Custom game trailer animation
- `images/` - Folder containing visual resources

## Local Launch

To view the site locally, simply open the `index.html` file in your web browser or run a local server.

## Technologies Used

- HTML5
- CSS3 (with advanced animations and effects)
- JavaScript (vanilla)
- Canvas API for the game trailer animation
- Font Awesome for icons
- Google Fonts for typography

## Note

This site is a demonstration and presents a fictional game. The download links are not functional.
